#  Movie Management System

A Java-based console application to manage movies.

## Features:
- Add, update, delete movies
- View all movies
- Search by:
- Name
- Category
- Rating range

## Technologies:
- Java
- OOP Concepts

## How to Run:
1. Compile the code
2. Run the main class
